The hangman game is a multiplayer game. In this game:
  - one player selects a word.
  - Other players have a certain number of guesses to guess the characters in the word.
  - If the players are able to guess the characters in the entire word within certain attempts, they win.
  - Otherwise, they lose.
  
So, Write a Program That:
  . The Computer chooses a word and Let the user know how many words it has
  . Then the user can guess the word
