# OOP Coffee Machine

This is a simple object-oriented programming (OOP) coffee machine implemented in Python.

## Usage

1. Make sure you have Python installed.
2. Clone this repository.
3. Run `python coffee_machine.py` to start the coffee machine.
4. Follow the on-screen instructions to make coffee.

## Secret Commands:
- off : turns off the coffee machine
- report: let's you know about how much money, water, coffee and milk the machine has.
